输入法框架选择与安装

Linux 中文输入法通常由三层组成：

flowchart TD
    A[应用程序] --> B[输入法框架：IBus / Fcitx5]
    B --> C[输入法引擎：Rime]
    C --> D[具体配置：雾凇拼音]

IBus 更适合 GNOME 桌面，因为 GNOME 对 IBus 集成较好。

Fcitx5 更适合 KDE、MATE、XFCE、LXQt、xrdp 等环境，中文输入体验和兼容性通常更好。
